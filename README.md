# Inventory Management System

מערכת לניהול מחסן מלאי, המאפשרת מעקב אחרי פריטים, כמויות, קטגוריות וספקים.

## מבוא

פרויקט זה נועד לספק בסיס ליישום ניהול מלאי עבור עסקים קטנים וגדולים. ניתן להשתמש בו כדי:

- להוסיף ולערוך פריטי מלאי
- לעקוב אחרי כמויות זמינות
- לנהל קטגוריות וספקים
- לקבל תצוגת מצב מלאי עדכנית

## התקנה

1. פתח טרמינל בתיקיית הפרויקט
2. התקן תלותיות (בהתאם לטכנולוגיה בשימוש):

```bash
# דוגמה עבור Node.js
npm install
```

או עבור Python:

```bash
pip install -r requirements.txt
```

## שימוש

הרץ את היישום בהתאם לסביבת העבודה שלך:

```bash
# דוגמה עבור פרויקט Node.js
npm start
```

```bash
# דוגמה עבור פרויקט Python
python main.py
```

## תכונות מוצעות

- ניהול מוצרים
- ניהול מלאי וערכי כמות
- חיפוש וסינון פריטים
- דוחות על רמות מלאי נמוכות
